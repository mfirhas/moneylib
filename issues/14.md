# Sub-issue 14: `#[allow(clippy::len_without_is_empty)]` on `ExchangeRates`

**Severity:** Low / Style

**File:** `src/exchange.rs` (line 461)

## Description

`ExchangeRates` suppresses the Clippy lint `len_without_is_empty` because no `is_empty()`
method is provided. However, `ExchangeRates` can never be empty — the base currency is always
present — so `is_empty()` could simply return `false`.

## Suggestion

Add an `is_empty() -> bool` method that unconditionally returns `false` (with a doc comment
explaining that the base currency is always present). This removes the need for the
`#[allow(...)]` suppression and makes `ExchangeRates` consistent with Rust's collection
conventions.
