# Sub-issue 9: `ExchangeRates<'a, Base>` has an unnecessary lifetime parameter

**Severity:** Medium

**File:** `src/exchange.rs` (line 253)

## Description

```rust
pub struct ExchangeRates<'a, Base: Currency> {
    rates: HashMap<&'a str, Decimal>,
    ...
}
```

Currency codes used as keys are always `&'static str` (they come from `C::CODE`). The `'a`
lifetime propagates into every type that holds `ExchangeRates`, requiring annotations like
`ExchangeRates<'static, USD>` in structs.

## Suggestion

Use `&'static str` directly (or `String`) as the key type to simplify the public API and
remove the unnecessary lifetime parameter from `ExchangeRates`.
